# MTG_Excel_listojen_kasittelya
Sattumanvaraisia python koodeja joilla käsittelen excel listoja

Download_Poromagia: Lataa poromagian kredutus sivuilta kaikki kredutettavat kortit exceliin, jotta niitä voi käsitellä (Epäoptimi mutta toimii))

Is_It_worth: Vertaa excel tiedoston sisällä paria arvoa ja suodattaa sieltä kortit, joita on kannattavaa tilata ulkomailta ja myydä suomen vähittäis myymälään (kesken)

formattingExcel: Pythonilla on helpompi formatoida kuin excelillä, erottaa kortin ja summan ja poistaa "X" niiden välistä

Mergedata: Poistaa wantti listalta kortteja joita on vaihdettu omistukseen
